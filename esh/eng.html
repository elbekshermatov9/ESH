~Please explain what you understand by computer programming.j

Also known as coding or programming, computer programming is the process of encoding an algorithm into a notation, typically a computer program, by means of some programming language so that it can be executed by a computer.

~Please explain the algorithm. What are some of its important features?

An algorithm can be defined as a set of final steps, the execution of which helps to complete a specific task. The important characteristics of an algorithm are clarity, efficiency, and finiteness.

~Please give a short explanation on the variables.

Variables are used to store program input as well as the results of calculations during program execution. In fact, these are named memory cells. The value stored in a variable may change during program execution.
